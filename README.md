# chat_application-project
Python Chat Application
 Project Description

This project is a simple real-time chat application built using Python and socket programming.
It follows a client–server architecture, where one program acts as a server and another acts as a client.
Users can send and receive messages through the command line in real time.

This project helps beginners understand how network communication works in Python.

 Features

Real-time messaging between two users

Client–Server communication model

Simple command-line interface

Uses Python socket programming

Lightweight and beginner-friendly project

 Technologies Used

Python
